A Unibit Starter Theme

Unibit is a superset of existing static site generators. It can be used as a standalone ssg or it can be imported into Stackbit and seamlessly converted into other static site generators and headless CMS.

Supported SSG
  • Jekyll
  • Hugo
  • Gatsby
Supported CMS
  • Forestry
  • Netlify CMS
  • DatoCMS
